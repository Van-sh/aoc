# AOC 2015

## Done in Gleam

to execute any solution, run the following command from `2025/` directory

```sh
gleam run -m day{day_number}/task{task_number}
# eg.
gleam run -m day01/task2
gleam run -m day10/task1
```

> [!NOTE]
>
> Unless I made a mistake, the paths commited in this repo should lead to `inputs/day{day_number}/input.txt` which is the actual input from AOC and therefore not commited, so either change the path before execution or make a file named `input.txt` in the respective inputs folder!
